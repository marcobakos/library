# library# library
Python library

### Setup environment

```
pip install virtualenv
git clone git@github.com:marcobakos/library.git
cd library
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
(optional) pip freeze
(optional) ls venv/lib/python3.7/site-packages/
```

